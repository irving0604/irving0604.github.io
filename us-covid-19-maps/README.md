# US COVID-19 Maps (2020)

## Project Overview
This project includes two interactive web maps showing COVID-19 data in the United States for the year 2020.  
The maps display county-level data using different thematic mapping techniques.

## Maps
- **COVID-19 Case Rates (Choropleth Map)**  
  This map shows COVID-19 case rates per 1,000 residents by county.  
  https://irving0604.github.io/us-covid-19-maps/map1.html

- **COVID-19 Case Counts (Proportional Symbols Map)**  
  This map shows total COVID-19 case counts by county using proportional circles.  
  https://irving0604.github.io/us-covid-19-maps/map2.html

## Data Sources
- COVID-19 case data: The New York Times  
- Population data: 2018 ACS 5-Year Estimates  
- County boundaries: U.S. Census Bureau  

The datasets were processed for use in this lab.

## Libraries Used
- Mapbox GL JS

## Features
- Interactive maps with click popups
- Choropleth map for COVID-19 case rates
- Proportional symbol map for COVID-19 case counts
- Legends, titles, and data sources included
- Albers projection used for both maps
